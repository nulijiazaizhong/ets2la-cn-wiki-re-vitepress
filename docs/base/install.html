准备工作 ​

1. 下载并安装C++ runtime ​

1.1. 下载C++ runtime ​

访问微软官方地址进行下载

1.2. 安装C++ runtime ​

双击下载下来的文件，先勾选我同意许可条款和条件，再点击安装。

如果你双击打开C++ runtime的安装程序是这样的，这表明你已经有了基础的环境，直接点击 关闭即可。

安装软件 ​

2.下载安装程序并完成安装 ​

2.1. 下载安装程序 ​

前往软件在GitHub上的发布页，下载最新地安装程序

提示

如果你无法通过GitHub下载到安装程序，我们同样提供镜像下载

:使用镜像下载的文件会在文件名中携带更新日期信息，例如：ETS2LA-Windows-Installer(update-10-10).exe

2.2. 运行安装程序 ​

双击运行刚才下载的安装程序，可能会出现Windows defender的提示，直接点击更多信息，展开之后选择仍要运行即可。
在完成上一步之后会来到选择语言的界面，选择Simplified Chinese并点击OK即可。

之后会进入到安装向导，点击下一步即可

点击我接受以同意许可协议

勾选我已知晓，若曾为此安装程序付费，即表示我已受骗并点击下一步

选择软件安装目录；建议使用盘符根目录以避免使用中文目录带来的无法安装的问题

仓库地址选择CNB,Pypi Mirror选择USTC或Aliyun，然后点击下一步

根据自己需要选择是否创建开始菜单快捷方式和桌面快捷方式，然后点击安装

安装过程中点击显示详情可以查看安装进度，点击显示详情也是在安装出现问题时最直接能够看出问题出现在什么地方的方式
当出现这样的内容时，稍作等待，最终需要的时间取决于各位的硬盘速度以及网络环境

安装完成后取消勾选启动ETS2LA，然后点击完成

提示

这里建议取消勾选启动ETS2LA，因为在启动ETS2LA时会下载一些文件，如果你的网络不好，可能会卡在下载文件上，导致每次启动都会出现警告；在后面的教程中 会教如何解决

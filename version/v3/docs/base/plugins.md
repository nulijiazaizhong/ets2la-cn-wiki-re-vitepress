# 插件介绍
这篇文章会以表格的形式来介绍ETS2LA`插件管理`页面中的所有插件实现的功能

## 基础模式 {#basic-plugins}
|        名称         |          作用          | 是否可选 |                      反作用                      |     备注     |
| :-----------------: | :--------------------: | :------: | :----------------------------------------------: | :----------: |
|      转向控制       |  使得车辆可以自动转向  |    ❌     |                   无法自动转向                   | 这是必开插件 |
|      速度控制       | 使得车辆可以自动加减速 |    ❌     |                  无法自动加减速                  | 这是必开插件 |
| Collision Avoidance |   在十字路口避免碰撞   |    ❌     | 无法在十字路口避免碰撞，可视化中右侧地图无法显示 | 这是必开插件 |
|         HUD         |   在游戏上层显示信息   |    ✔     |                无法在游戏上层显示                | AR插件为基础 |
|     文本转语音      |  提示信息使用语音播报  |    ✔     |               提示信息无法语音播报               |   可选插件   |
| 环境可视化 | 能够在环境可视化页面查看信息 | ✔ | 无法在环境可视化中查看信息 | 环境可视化页面无法显示，推荐打开 |

## 高级模式 {#advaced-plugins}

|         名称          |                     作用                     | 是否可选 |                      反作用                      |             备注              |
| :-------------------: | :------------------------------------------: | :------: | :----------------------------------------------: | :---------------------------: |
|    Event Listener     |                 监听游戏信息                 |    ❌     |                   整个插件失效                   |     默认开启的插件，别关      |
|  Collision Avoidance  |              在十字路口避免碰撞              |    ❌     | 无法在十字路口避免碰撞，可视化中右侧地图无法显示 |         这是必开插件          |
|    自适应巡航控制     |          根据道路信息自动控制加减速          |    ❌     |                无法自动控制加减速                |         这是必开插件          |
|         地图          |              提供转向和导航功能              |    ❌     |          无法获取游戏信息，基础功能失效          |         这是必开插件          |
|         导航          |              获取游戏内导航信息              |    ❌     |        无法获取游戏导航信息，上述插件失效        |         这是必开插件          |
|        可视化         |         能够在环境可视化页面查看信息         |    ❌     |            无法在环境可视化中查看信息            |         这是必开插件          |
|          TTS          |             提示信息使用语音播报             |    ✔     |               提示信息无法语音播报               |           可选插件            |
| Discord Rich Presence |          在Discord上显示ETS2LA状态           |    ✔     |            Discord上不显示ETS2LA状态             |          需网络支持           |
|          HUD          | 在游戏上层显示信息添加覆层，为其余插件做基础 |    ✔     |   无法在游戏上层显示没有覆层，无法显示多余信息   | AR插件为基础HUD之类的插件失效 |
|          AR           |          添加覆层，为其余插件做基础          |    ✔     |            没有覆层，无法显示多余信息            |       HUD之类的插件失效       |
::: warning 警告
`Discord Rich Presence` 插件仅支持Discord客户端在打开的情况下显示，网页不支持
:::
